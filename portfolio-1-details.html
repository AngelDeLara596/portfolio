Personal Portfolio Website

This website encapsulates my professional journey by presenting a comprehensive showcase of my expertise as a Frontend Web Developer, including an interactive resume and an extensive portfolio highlighting diverse projects. Crafted meticulously using Bootstrap, this platform incorporates a sleek, minimalist design adorned with seamless animations for an elegant user experience.

This website embodies responsiveness, seamlessly adapting to diverse screen sizes to ensure optimal viewing on any device. In addition, it offers a thoughtful feature—a dark mode toggle—to enhance accessibility and user comfort.

With a commitment to modern design principles and user-centric functionality, this project is a testament to my proficiency in front-end development, reflecting both technical prowess and a keen eye for aesthetic appeal.
